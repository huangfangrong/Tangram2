top使用说明

baidu.fx.collapse()

语法
baidu( selector[, context] ).collapse( options] );
baidu.fx( jQuery|$DOM|NodeList ).collapse( options] );
baidu.fx.collapse( elementoptions] );
参数说明
selector

CSS选择器字符串

context

[可选]指选择器的范围

jQuery

通过jQuery的 $() 取得的DOM元素集合对象

$DOM

通过Tangram的 baidu() 或 baidu.dom() 取得的DOM元素集合对象

NodeList

通过 document.getElementsByTagName() 或者 dom.childNodes 或者 document.images 等方式取得的DOM元素集合

element

DOM对象

options
名称 类型 含义 取值
startOrigin 属性 遮罩效果的起始点 "0px 0px" 或 "50% 50%" 或 "top left"
beginColor 属性 起始的背景色 #FFFF00
endColor 属性 结束时的背景色 #FFFFFF
finalColor 属性 最终呈现的背景色 #FFFFFF
textColor 属性 文本颜色 #000000
duration 属性 动画时长 正整数（单位：毫秒）
delay 属性 延时指定时长再开始动画 正整数（单位：毫秒）
delta 只读属性 动画两帧之间的进度 小数
percent 只读属性 动画进度的百分比值 小数
interval 属性 两帧之间的最小间隔时间 正整数（毫秒）
fps 属性 动画最大帧率 正整数（单位：次/秒）
frames 属性 动画至少帧数，可能会延时保帧 正整数
from 属性 动画起始的进度 0-1 的小数（包括0和1）
to 属性 动画终止的进度 0-1 的小数（包括0和1）
transtion() 方法 动画的线型函数 function(percent){return percent}
onbeforestart 事件 在动画运行之前触发 function
onstart 事件 在动画运行之后触发 function
onupdate 事件 动画过程中，持续触发 function
onfinish 事件 动画准备结束时触发 function
onafterfinish 事件 动画结束后触发 function
ondispose 事件 动画被析构时触发 function
onabort 事件 动画被强行中断触发 function
onend 事件 动画被直接结束时触发 function
oncancel 事件 动画被取消时触发 function
onpause 事件 动画被暂停时触发 function
动画的操作方法
名称 类型 含义
play() 方法 开始执行动画，或者重启被暂停的动画
pause() 方法 暂停动画
abort() 方法 强行中断动画
cancel() 方法 取消动画
end() 方法 直接结束
二次开发
名称 类型 含义
baidu.fx.create(element, methods, fxName) 方法 通过这个方法可以快速创建动画
methods里可传递5个方法：
1、initialize()
2、render(percent)
3、finish()
2、restore()
2、transition(percent)

create()方法对 Timeline 类扩展了：
protect(cssKey)
restoreAfterFinish
两个属于/方法
baidu.fx.extend({fx}) 方法 该方法会创建
静态方法：baidu.fx.fxName()
链式调用：baidu.fx().fxName()
DOM链式：baidu().fxName()
baidu.fx.Timeline([options]) 类 属性/方法/事件 参考 options
